# ADR-0003: 5DOF 초기 작업 제약

- 상태: 채택
- 날짜: 2026-07-12

## 상황

SO-ARM101은 자세를 만드는 관절 5개와 gripper 1개로 구성된다. 따라서 임의의 6차원 말단 장치(end-effector) 자세를 항상 만들 수는 없다.

## 결정

1차 Pick and Place는 평평한 작업대를 위에서 내려다보며 접근하는 방식(top-down)으로 제한한다.

- 위치 `x, y`와 접근 높이를 우선한다.
- 허용되는 범위에서 펜의 평면 yaw를 맞춘다.
- 펜꽂이는 초기에는 입구가 넓은 것을 사용한다.
- 도달할 수 없는 자세는 IK 대체 계산으로 억지 실행하지 않고 작업 실패로 처리한다.
